# 학생 성적 관리 앱
- MVC모델
- SQLite를 사용하여 기존에 저장 된 학생 또는 새로운 학생을 추가할 수 있다.
- 학생의 점수를 추가하고 점수를 JS를 통해 그래프로 확인할 수 있다.
- 학생이 많아질 경우를 대비해 검색 기능도 가지고있다.
